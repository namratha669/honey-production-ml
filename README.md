# 🍯 Honey Production Prediction (Machine Learning)

This project uses Linear Regression to analyze and predict the decline in honey production in the United States.


## 📊 Project Overview

Honeybee populations have been declining, affecting honey production.
This project analyzes historical data and predicts future production trends up to 2050.

---

## 🛠️ Tech Stack

* Python 🐍
* Pandas
* NumPy
* Matplotlib
* Scikit-learn

---

## 📈 What This Project Does

* Analyzes honey production data by year
* Builds a Linear Regression model
* Visualizes trends using graphs
* Predicts future honey production (2013–2050)

---

## 📉 Results

* The model shows a **declining trend** in honey production
* Predicted production in 2050 is significantly lower than past values

---

## ▶️ How to Run

```bash
pip install -r requirements.txt
python main.py
```
